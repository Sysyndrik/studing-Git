Удаление веток происходит при помощи команды git branch -d <имя ветки>. Этот ключ выдаст предупреждение, если в ветке находятся изменения, которые не «слились» с основной веткой. Если хотите удалить без предупреждений, тогда нужно использовать ключ -D. 8. Самостоятельно создайте новый файл и поместите его на ветку feature, посмотрите изменения на GitHub.
